## PoC 範圍（v0.1）

### 核心功能

- [x] 時間軸主畫面
- [x] 儀表板（飲水/餐數/吸菸）
- [x] 統一輸入入口（AI 自動辨識）
- [x] 晨間/睡前儀式
- [x] AI 諮詢（多輪對話）
- [x] 近 7 天報告
- [x] 目標管理
- [x] 匯出（CSV + Skill Prompt）
- [x] API 設定

### 支援的輸入類型

- [x] 飲食（文字/照片/混合）
- [x] 飲水
- [x] 運動（跑步、跑步機、健行、步行、騎乘）
- [x] 步數/樓層數
- [x] 血糖（單點/整天）
- [x] 血壓
- [x] 體重/體脂
- [x] 睡眠
- [x] HRV
- [x] 簡易事件（吸菸、壓力、情緒）

### 追蹤功能

- [x] Analytics Sheet（input_save、ai_consult、ai_weekly_review）
